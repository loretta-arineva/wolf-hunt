## About this project

Wolf Hunt is a simple Doodle Jump inspired jumping game. You can [play it online now](https://loretta-arineva.github.io/wolf-hunt/).

## Technologies

 * HTML5
 * CSS3 
 * Flexbox
 * Vanilla JS

## How to play

 * **Move left** - Left arrow 
 * **Move right** - Right arrow
 * **Move straight up** - Arrow up
